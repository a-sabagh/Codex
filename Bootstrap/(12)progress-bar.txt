progress-bar:
To create a default progress bar, add a .progress class to a <div> element:
	.progress
	.progress-bar
	style="width=n%;"
=================================================
this attrebute is added to div.progress-bar
	role="progressbar"
	aria-valuenow="n"
	aria-valuemin="0"
	aria-valumax="100"
=================================================
this class is remove the text in progressbar
	span.sr-only
=================================================
Colored Progress Bars:
    .progress-bar-success
    .progress-bar-info
    .progress-bar-warning
    .progress-bar-danger
=================================================
Striped Progress Bars:
	.progress-bar-striped
=================================================
Animated Progress Bar:
	.active
=================================================
